## React To-Do App, powered by Deno

This app is a POC for a Microservice (CRUD) for a to-do list.

**Technology**

-   Frontend: React
-   API: Deno, Typescript
-   Infrastructure: Docker, Docker Swarm
-   Database: MongoDB

**Build & Run Swarm**

```
docker-compose -f stack.yml up --build
```

**Run Swarm**

```
docker-compose -f stack.yml up
```
